# Mini Calculator 🧮

Mera Python ka calculator hai ye!

Ye chota sa project do number jodta, ghata, guna karta aur bhag karta hai.

### Kaise chalaye:
1. `calculator.py` file download karo
2. Terminal/CMD kholo 
3. `python calculator.py` likho
4. Enter dabao - Ho gaya! ✅

### Kya kya karta hai:
+ Jodta hai
- Ghatata hai  
* Guna karta hai
/ Bhag karta hai
